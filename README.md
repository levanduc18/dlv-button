# test-vue-button

## Publish library to npm:

```
Hiện tại đã publish library lên npm với module có tên dlv-button@0.1.5 (https://www.npmjs.com/package/dlv-button):
Module rất đơn giản, có 1 VueButton nền xanh dương nhạt, và event onClick.
Các bước để publish 1 npm lên server npm (nếu cần):
- Step 1: chạy lệnh yarn build:l;
- Step 2: npm publish --access public.
```
